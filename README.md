# Akwa-Dialysis
## Description
“Smart Pre-Dialyzer” is a monitored dialysate circuit using mobile application and Web Site.  
“Smart Pre-Dialyzer” simulates Dialysate Circuit and its Alarm and Monitoring systems.
## Used Tools
Arduino  
Flutter
